# threejs-act3.4-3.6

## Project Setup (for all activities)

# Make sure you have:
- Node.js installed
- VS Code (or another code editor)

## Installation & Running
Open the folder of the activity you want to run (e.g., Activity 3.6 Raycaster and Mouse Events)

# Open a terminal in that folder
# Run the following commands:
npm install
npm run dev
