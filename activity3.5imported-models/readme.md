This activity demonstrates how to import, display, and animate 3D models in Three.js using the GLTFLoader and DRACOLoader.

## Learning Objectives
# Understand different 3D model formats (GLTF, GLB, Embedded, Draco).
# Learn how to import models using GLTFLoader.

# Implement DRACOLoader for compressed models.
Apply lights, scaling, and animations to imported models.

Explore AnimationMixer for playing model animations.
